Cybersecurity Glossary

A comprehensive list of common cybersecurity terms and their definitions.

A B C D E F H I K L M P R S T V W Z

A

Antivirus Software

Software designed to detect, prevent, and remove malicious software from a computer system.

B

Backdoor

A secret method of bypassing normal authentication or encryption in a computer system, product, or device.

Biometric Authentication

Security process that relies on the unique biological characteristics of individuals to verify their identity.

Blacklist

A list of entities that are denied access to a particular privilege, service, or resource.

Botnet

A network of private computers infected with malicious software and controlled as a group without the owners' knowledge.

Brute Force Attack

An attack that attempts to gain unauthorized access by systematically trying all possible combinations of passwords or encryption keys.

C

CAPTCHA

Completely Automated Public Turing test to tell Computers and Humans Apart - a challenge-response test used to determine if the user is human.

Cookie

A small piece of data stored on a user's computer by websites to remember information about the user.

Cryptography

The practice and study of techniques for secure communication in the presence of third parties.

D

Data Breach

An incident where sensitive, protected, or confidential data is accessed, viewed, stolen, or used by an unauthorized individual.

DDoS Attack

Distributed Denial of Service attack - an attempt to make an online service unavailable by overwhelming it with traffic from multiple sources.

Digital Certificate

An electronic document that proves the ownership of a public key, used to verify the identity of websites and secure communications.

Digital Signature

A mathematical scheme for verifying the authenticity of digital messages or documents.

E

Encryption

The process of converting data into a code to prevent unauthorized access. Only authorized parties can decrypt and read the data.

End-to-End Encryption

A method of secure communication that prevents third parties from accessing data while it's transferred from one end system to another.

F

Firmware

Software that provides low-level control for a device's specific hardware.

Firewall

A network security device that monitors and filters incoming and outgoing network traffic based on predetermined security rules.

H

Hashing

The process of converting data into a fixed-size string of characters, which is typically a hash value or hash code.

Honeypot

A security mechanism that creates a virtual trap to lure attackers and detect, deflect, or study hacking attempts.

I

Intrusion Detection System (IDS)

A device or software application that monitors a network or systems for malicious activity or policy violations.

IP Address

A unique numerical label assigned to each device connected to a computer network that uses the Internet Protocol for communication.

K

Keylogger

Malware that records every keystroke made on a computer's keyboard, often used to steal passwords and other sensitive information.

L

MAC Address

A unique identifier assigned to network interfaces for communications at the data link layer of a network segment.

M

Malware

Malicious software designed to harm or exploit any programmable device, service, or network. Examples include viruses, worms, trojans, and ransomware.

Man-in-the-Middle Attack

An attack where the attacker secretly relays and possibly alters the communications between two parties who believe they are directly communicating with each other.

Multi-Factor Authentication (MFA)

A security system that requires multiple methods of authentication from independent categories of credentials to verify the user's identity.

P

Packet Sniffing

The practice of capturing and analyzing network traffic to monitor or intercept data packets.

Password Manager

A software application that helps users generate, store, and manage their passwords for various online services.

Patch

A set of changes to a computer program or its supporting data designed to update, fix, or improve it.

Penetration Testing

An authorized simulated attack on a computer system to evaluate its security.

Phishing

A type of social engineering attack where attackers impersonate legitimate entities to trick users into revealing sensitive information or downloading malware.

Public Key Infrastructure (PKI)

A set of roles, policies, and procedures needed to create, manage, distribute, use, store, and revoke digital certificates.

Proxy Server

A server that acts as an intermediary for requests from clients seeking resources from other servers.

R

Ransomware

Malware that encrypts a victim's files and demands payment (ransom) to restore access to the data.

Risk Assessment

The process of identifying, analyzing, and evaluating potential security risks to an organization's assets.

Rootkit

Malware that provides privileged access to a computer while hiding its existence from users and security programs.

S

Sandbox

An isolated environment where untrusted programs can be executed safely without affecting the rest of the system.

Security Policy

A set of rules and practices that specify how an organization manages, protects, and distributes sensitive information.

Social Engineering

The psychological manipulation of people into performing actions or divulging confidential information.

Spyware

Software that secretly collects information about a user's activities without their knowledge or consent.

SSL/TLS

Secure Sockets Layer/Transport Layer Security - protocols that provide communications security over a computer network.

T

Token

A physical or digital device used to authenticate a user's identity, often used in two-factor authentication.

Two-Factor Authentication (2FA)

A security process that requires users to provide two different authentication factors to verify their identity.

V

VPN (Virtual Private Network)

A service that creates a secure, encrypted connection over a less secure network, such as the internet.

Vulnerability

A weakness in a system that can be exploited by attackers to perform unauthorized actions.

W

Whitelist

A list of entities that are granted access to a particular privilege, service, or resource.

Worm

A standalone malware computer program that replicates itself to spread to other computers.

Z

Zero-Day Exploit

A cyber attack that occurs on the same day a weakness is discovered in software, before a fix becomes available.

Zombie Computer

A computer connected to the internet that has been compromised by a hacker and can be used to perform malicious tasks.
